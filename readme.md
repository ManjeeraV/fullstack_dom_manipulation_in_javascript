The Portfolio Website is created using HTML, CSS and JavaScript. The concepts of DOM manipulations are used in this project.


The details of the project are as follows :
● The website contains a single page.

● The web page has three sections:
 o About Me: Here the website owners will mention information about themselves.  
 o Enter New Project Details: Here they can enter details about the new project they want to add to their portfolio. 
 Once they click on the save button, the project will be added to the existing projects on the webpage.
 o My Projects: Here they can display their projects with details like project title, description and image.
 
● You will find the following files in the folder: 
 o index.html: This file contains the html for the webpage.
 o styles.css: This file contains the styling for index.html.
 o projects.js: This file contains an array of projects.The array consists of a collection of objects. Each object consists of details regarding a project such as title, description and image.
 o script.js: In this file, you have to add code to complete the tasks which are mentioned in the Problem Statement.

 To run the application on your default browser, double click the index.html file.
